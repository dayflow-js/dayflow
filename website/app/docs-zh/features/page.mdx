# 功能展示

这一章节汇集了所有可交互的演示，帮助你在动手集成前快速了解 DayFlow 能提供的体验。

- **视图切换器** – 在按钮/下拉两种模式间切换，比较不同布局在桌面与移动端的体验。
- **自定义事件详情面板/对话框** – 学习如何只替换面板内容、或直接提供自定义模态，实现审批、颜色选择等流程。
- **事件对话框** – 体验内置 CRUD 表单，了解时间范围选择、全天转换、颜色选择、备注输入等细节。
- **侧边栏** – 在默认布局与自定义布局间切换，观察 KPI、过滤器等内容如何与日历状态保持同步。
- **深色模式** – 测试浅色/深色/自动切换，了解如何针对不同日历类型配置独立的浅/深色配色。

从左侧侧边栏挑选任意主题，配合源码与演示，一起探索 DayFlow 的定制潜力。
